전체 선택자 : * { color: red; }

태그 선택자 : dt { color: red; }

아이디 선택자 유일 : #h1 { color: red; }

클래스 선택자 중복 가능, 여러게 가능 : .red { color: red; }

기본 속성 선택자 : [class] { color: red; }

기본 속성 선택자 : [class='red'] { color: red; }

기본 속성 선택자 : [href='#'] { color: red; }

그룹 선택자 : h1, p { color: red; }

자식 선택자 : div > h2 { color: red; }

하위 선택자 : div h2 { color: red; }

인접 형제 선택자 : h2 + h3 { color: red; }

일반 형제 선택자 모두 선택 : h2 ~ h3 { color: red; }

가상요소 선택자 : h2::before 또는 h2::after { content: 'test 문자열'; }

가상클래스 선택자 요소의 상태 a 태그 방문 : a:link 또는 a::hover 또는 a::active { color: red; }

입력요소 가상클래스 선택자 input : input:focus 또는 input::checked + label { color: red; }
